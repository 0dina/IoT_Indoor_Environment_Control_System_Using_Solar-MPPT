# IoT_Indoor_Environment_Control_System_Using-Solar_MPPT

• Built IoT-based smart window system integrating 4 sensors (temperature, humidity, fine dust, CO) for real-time monitoring.
• Implemented SQL database logging, managing >5,000 sensor entries/day and tracking window operation logs.
• Designed web UI with Chart.js, HTML, CSS, JavaScript for real-time visualization and enabled remote control with <1s latency.
